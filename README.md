# EduPlatforma

Компания ЭдуПлатформа занимается предоставлением сервисов для:

1.*Колледжей* 

2.*Профильных учебных заведений*

Которые рассылает задания и фиксирует срок сдачи с требованиеми заданными преподавателем

Распеределяет нагрузку и составляет равномерной расписание занятий в соответсвии с доступностью

Требуется:
1. Создать информационную систему

2. Провести интегрирование с существующми академическими БД
